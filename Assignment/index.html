DESCRIPTION


This assignment covers finding the euclidean distance between 2 pixels. To make it more challenging couple of things are included. First of all identification of objects is done. The primary purpose is to arrange the (x, y)-coordinates associated with a rotated bounding box in top-left, top-right, bottom-right, and bottom-left order. Organizing bounding box coordinates in such an order is a prerequisite to performing operations such as matching corners of objects as we compute the distance between objects.

Then edge detection is done for all the objects and we find the distance between the reference object and all the other objects (here the reference object is first object on leftmost side). Now taking reference object as the base object we find the distance of other objects from the reference object. Every distance is displayed on the screen itself to make it look more interesting, you can press ENTER key and see the next distance measure between the pixels. Also the pixel distance is converted into inches to make it more human readable. So enjoy finding the distances between the objects inside the given image using the given code.
